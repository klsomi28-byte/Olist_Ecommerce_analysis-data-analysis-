# Olist_Ecommerce_analysis-data-analysis-
End -to-End E-commerce data analysis using python as MySQL
Project Overview

This project presents an end-to-end analysis of the Brazilian Olist E-Commerce dataset using Python and SQL. The analysis focuses on extracting meaningful business insights related to revenue generation, payment trends, order patterns, and delivery performance.

The project demonstrates practical implementation of data cleaning, SQL querying, data visualization, and business insight extraction.

 Key Insights
The business generated over ₹13.59 million in total revenue.

Credit Card is the most preferred payment method among customers.

Approximately 6.77% of deliveries were delayed.

Revenue peaked in 2018, indicating strong business growth.

Monthly order trends show seasonal variations.
